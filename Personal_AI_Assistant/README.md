# Personal AI Assistant

第一位個人 AI 助理的工作空間。

## 一句話目的

協助整理筆記、起草文字草稿，以及 AI Agent 入門培訓的學習練習。

## 允許做的事

- 整理培訓筆記與學習摘要
- 起草 email 或訊息草稿（使用假名／範例資料）
- 練習 prompt 撰寫與 Agent 工作流程
- 建立非機密的參考資料與 checklist

## 禁止做的事

- 存放或要求提供：密碼、API key、token、憑證檔
- 存放或討論：薪酬、員工紀錄、客戶個資
- 貼上真實內部連結、未公開商業機密
- 執行或建議：delete、force、admin permission、credential 相關指令

## Session 規則

- 每次練習在 `sessions/` 建立一個檔案或子資料夾
- 命名格式：`YYYY-MM-DD_主題`（例如 `2026-07-07_training-intro.md`）
- 練習內容只用假名與範例資料
- 若需執行 terminal command，先解釋用途，再確認後執行
